Arrastar e Soltar em HTML - Tutorial

Como Implementar o Drag and Drop

Para implementar o arrastar e soltar em HTML, você pode usar os eventos `dragstart`, `drag`, `dragover` e `drop`. Veja um exemplo abaixo:


Exemplo de Código

<div class="dropzone" ondrop="drop(event)" ondragover="allowDrop(event)">
  Arraste e solte aqui!
</div>
<div class="dragged-element" draggable="true" ondragstart="drag(event)">Elemento Arrastável</div>

Como auxilio coloquei também css:

.dropzone {
width: 300px;
height: 200px;
border: 2px dashed #ccc;
text-align: center;
line-height: 200px;
font-size: 20px;
color: #666;
margin: 20px auto;
}


.dragged-element {
width: 100px;
height: 100px;
background-color: #f00;
color: #fff;
text-align: center;
line-height: 100px;
font-size: 16px;
position: absolute;
top: 0;
left: 0;
display: none;
}
