Formatting

Elementos de formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

<b>- Texto em negrito

<strong>- Texto importante

<i>- Texto em itálico

<em>- Texto enfatizado

<mark>- Texto marcado

<small>- Texto menor

<del>- Texto excluído

<ins>- Texto inserido

<sub>- Texto subscrito

<sup>- Texto sobrescrito
