Mídia no HTML - Tutorial

O que é Mídia no HTML?

Os elementos de mídia no HTML permitem incorporar e exibir diferentes tipos de conteúdo de mídia, como áudio, vídeo e imagens, diretamente em uma página da web. Isso pode incluir arquivos de áudio MP3, vídeos MP4, imagens JPEG, entre outros.


Exemplo de código para inserir um vídeo

<video width="400" height="300" controls>
  <source src="video.mp4" type="video/mp4">
 Seu navegador não suporta o elemento <video>.
</video>
