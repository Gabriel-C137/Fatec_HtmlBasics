Headings de tags HTML

Títulos maiores

Cada título HTML possui um tamanho padrão. No entanto, você pode especificar o tamanho de qualquer título com o style atributo, usando a font-size propriedade CSS:

<h1 style="font-size:60px;">Heading 1 </h1>

Ficará assim:

Heading 1

Cabeçalhos HTML

Os títulos HTML são definidos com as tags <h1> to <h6>.<h1>define o título mais importante. <h6> define o título menos importante.

Muito importante usar títulos para mostrar a estrutura do documento.
