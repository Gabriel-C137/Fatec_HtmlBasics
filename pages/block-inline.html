Elementos Block e Inline HTML

O que são Elementos Block e Inline HTML?

Elementos de bloco (block) são elementos HTML que ocupam toda a largura disponível e geralmente começam em uma nova linha. Já os elementos em linha (inline) ocupam apenas o espaço necessário e não iniciam uma nova linha.
