Elementos de Bloco vs Elementos Inline

Em HTML, elementos podem ser classificados como elementos de bloco ou elementos inline, dependendo de como eles são exibidos no layout da página.

Elementos de Bloco

Elementos de bloco ocupam todo o espaço horizontal disponível e geralmente começam em uma nova linha. Alguns exemplos comuns de elementos de bloco são: `

Elementos Inline
