Entidades HTML

O que são Entidades HTML?

Entidades HTML são códigos especiais que representam caracteres especiais, como símbolos, emojis e caracteres que têm significados especiais em HTML, como `<`, `>`, `&`, entre outros.

Exemplo de Uso de Entidades HTML:

Este é o símbolo de direitos autorais: © (& copy;)


Este é um emoji de coração: 💗 (Cógigo do Emoji)


Para Acessar o código de diversos emojis, clique aqui

Entidades HTML são úteis quando você precisa exibir caracteres especiais que não podem ser facilmente digitados ou que têm significados especiais em HTML.
