Div em HTML - Tutorial

O que é a tag <div>?

A tag `

` é um elemento de bloco que é usado para agrupar outros elementos HTML em uma seção específica ou para criar layouts mais complexos em uma página da web. A tag `
` por si só não tem nenhum significado semântico, mas é frequentemente usada em conjunto com classes ou IDs CSS para estilizar e organizar o conteúdo.

Exemplo de Código

<div class="div-example">
  <p>Este é um exemplo de conteúdo dentro de uma div.</p>
</div>
