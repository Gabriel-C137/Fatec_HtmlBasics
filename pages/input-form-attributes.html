Atributos de Formulário para Campos de Entrada em HTML

O que são Atributos de Formulário para Campos de Entrada em HTML?

Atributos de Formulário para Campos de Entrada em HTML são propriedades que podem ser adicionadas aos elementos de entrada, como <input>, para especificar em qual formulário o campo deve ser incluído.
