JavaScript no HTML

O que é JavaScript no HTML?

O JavaScript embutido em HTML permite adicionar scripts diretamente no código HTML, o que pode ser útil para realizar tarefas simples, como manipulação do DOM (Modelo de Objeto do Documento) ou execução de funções específicas.

O JavaScript embutido em HTML é uma maneira rápida e conveniente de adicionar interatividade a uma página da web, mas é mais comum separar o código JavaScript em arquivos externos para facilitar a manutenção e o gerenciamento do código.
