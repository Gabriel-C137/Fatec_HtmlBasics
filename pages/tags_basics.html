4.

Links para sua página

<a href=""> - Aqui você pode aplicar um link associado á um texto


<a href="">Link para site externo</a>

a - link, href - refere-se

Link para site externo

5.

Links para sua página

<img src="" alt=""> - Aqui você pode aplicar um link associado á um texto


<img src="img/joia.png" alt="emoji de joia">

img - identifica imagem, src - refere-se, alt - é uma descrição textual da imagem
