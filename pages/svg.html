SVG em HTML - Tutorial

O que é SVG?

SVG (Scalable Vector Graphics) é uma linguagem de marcação baseada em XML para descrever gráficos vetoriais bidimensionais. Em contraste com imagens rasterizadas (como JPEG, PNG, GIF), que representam imagens como uma grade de pixels, os gráficos vetoriais são compostos de formas geométricas primitivas, como linhas, curvas e formas, que são renderizadas com precisão, independentemente do tamanho em que são exibidas.


Exemplo de código SVG

<svg width="400" height="200">
  <circle cx="100" cy="100" r="50" fill="red" />
  <rect x="200" y="50" width="100" height="100" fill="blue" />
  <text x="150" y="180" font-size="24" fill="green">SVG</text>
</svg>
