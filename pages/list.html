Listas HTML

O que são Listas HTML?

Listas HTML são utilizadas para organizar informações em formatos de itens numerados ou itens não numerados.

Exemplo de Lista Não Numerada:

  • Item 1
  • Item 2
  • Item 3

Exemplo de Lista Numerada:

  1. Item 1
  2. Item 2
  3. Item 3

As listas HTML podem ser criadas usando os elementos <ul> para listas não numeradas e <ol> para listas numeradas. Cada item da lista é representado pelo elemento <li>.
