Listas em HTML - Tutorial

Como Criar Listas em HTML

Existem dois tipos principais de listas em HTML: listas não ordenadas (`

    `) e listas ordenadas (`
      `). Aqui está como criar cada uma:


      Exemplo de Código para Listas Não Ordenadas

      <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
      </ul>

      Exemplo de Código para Listas Ordenadas

      <ol>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
      </ol>


Exemplo de Lista Não Ordenada

  • Item 1
  • Item 2
  • Item 3

Exemplo de Lista Ordenada

  1. Item 1
  2. Item 2
  3. Item 3
