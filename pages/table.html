Tabelas em HTML - Tutorial

Como Criar Tabelas em HTML

O elemento HTML `

` é usado para criar tabelas em uma página da web. Aqui está um exemplo básico de como criar uma tabela:


Exemplo de Código

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>João</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Maria</td>
    <td>25</td>
  </tr>
</table>


Exemplo de Tabela

Nome Idade
João 30
Maria 25
