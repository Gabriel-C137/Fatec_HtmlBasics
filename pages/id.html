IDs HTML

O que são IDs HTML?

IDs em HTML são atributos que podem ser adicionados a elementos HTML para identificá-los de forma única em uma página.

Podemos aplicar estilos específicos ou comportamentos JavaScript a elementos com IDs únicos.


Os IDs HTML devem ser únicos em uma página, o que os torna úteis para identificar elementos específicos para estilização ou manipulação de script.
