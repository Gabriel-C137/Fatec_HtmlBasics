IDs em HTML - Tutorial

O que são IDs em HTML?

O atributo "id" em HTML é usado para identificar exclusivamente um elemento na página da web. Cada ID deve ser único dentro de um documento HTML e pode ser usado para estilizar, acessar ou manipular o elemento usando JavaScript ou CSS.

Exemplo de Código

<div id="content">
  <p>Este é um conteúdo com um ID específico.</p>
</div>
